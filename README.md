# Mutual Fund Recommendation & Investment Analytics Dashboard - Power BI

## Project Overview

This project is an interactive Power BI dashboard developed for mutual fund analysis and investment recommendation.

The dashboard helps users analyze mutual fund performance, risk levels, fund categories, and investment options based on different investor preferences.


## Business Objective

The objective of this dashboard is to transform mutual fund data into meaningful insights that help users compare funds and make informed investment decisions.


## Dashboard Features


## 1. Investment Return Analysis

The dashboard provides analysis of:

- 1 Year Return Performance
- 3 Year Return Performance
- 5 Year Return Performance

Includes:

- Return percentage analysis
- Fund type distribution
- Risk level comparison


## 2. Mutual Fund Recommendation System

Provides fund recommendations based on user preferences:

- Investment goal
- Risk appetite
- Investment duration
- Fund type
- Market knowledge
- Liquidity requirements
- ELSS preference


## 3. Fund Performance Analysis

Includes:

- Fund wise NAV analysis
- Fund category comparison
- AMC wise analysis
- Equity, Debt and Hybrid fund comparison


## Key Insights

- Comparison of mutual fund categories
- Identification of high performing funds
- Risk based investment analysis
- Personalized fund filtering
